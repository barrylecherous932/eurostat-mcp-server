# 0.1.1 — 2026-05-23

## Added

- **`eurostat_search_datasets`** — keyword search across 8,933 Eurostat datasets using a session-cached TOC TXT file
- **`eurostat_browse_themes`** — hierarchical navigation of the Eurostat theme tree (11 root themes)
- **`eurostat_get_dataset_info`** — dataset metadata: dimensions, valid values, time range, obs count, last-updated date
- **`eurostat_get_dimension_values`** — full value list for a specific dataset dimension; supports NUTS geo-level filtering
- **`eurostat_query_dataset`** — decoded JSON-stat 2.0 observations with dimension filters, NUTS geo-level, and time-range support
- **`eurostat://dataset/{dataset_code}`** resource — dataset metadata accessible by URI
- `EurostatCatalogueService` — fetches and parses the TOC TXT file with session-level in-memory cache
- `EurostatDataService` — Statistics API HTTP client with JSON-stat 2.0 stride decoder, async-response polling, and retry logic
- `EUROSTAT_BASE_URL` env var — override API base URL (default: `https://ec.europa.eu/eurostat/api/dissemination`)
- `EUROSTAT_REQUEST_TIMEOUT_MS` env var — HTTP request timeout in ms (default: `30000`)
- 37 tests across all tools and the dataset resource
