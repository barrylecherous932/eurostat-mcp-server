# eurostat-mcp-server

EU-wide statistics via the Eurostat JSON API.

## Data source

- **Eurostat JSON API** — EU statistical data across economy, demographics, trade, labor, environment, digital economy
- **Auth**: None required
- **Rate limits**: Generous, public API

## Why it earns its keep

Fills the European data gap alongside World Bank (global) and BLS/FRED (US) for three-tier economic coverage. NUTS-region granularity gives sub-national detail unavailable in World Bank. 27 EU member states + candidate countries.

## Target users

- Economic researchers comparing EU countries or regions
- Journalists covering EU policy and economics
- Business analysts evaluating European markets
- Agents doing cross-regional analysis (pair with BLS for US, World Bank for global)
- Policy researchers working on EU topics

## Scope

- Read-only
- Dataset/table discovery and search
- Data queries with dimension filters (country, time, indicator)
- NUTS region hierarchy browsing
- Dataset metadata and dimension values
- Table of contents / theme navigation
